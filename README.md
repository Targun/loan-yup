# loan-yup
Simple module for validating loan applications

## INSTALL
Install via NPM:

```bash
npm install loan-yup

```

## USAGE

#### javascript

```javascript

var loanYup = require("loan-yup");

loanYup.formSchema().validate(obj);

```

#### TypeScript

```typescript

import * as loanYup from 'loan-yup';

loanYup.formSchema().validate(obj);


```
